Selector css

  1. Universal Selector
    2. Universal selector merupakan selector yang sering programmer gunakan untuk
    memilih dan menyeleksi semua elemen pada suatu dokumen HTML.
    contoh kode :
  2. Tag Selector
    3. Sesuai dengan namanya, selektor tag merupakan selektor yang memilih
    elemen berdasarkan nama tagnya. Selektor ini juga bisa kita sebut Type Selector.
    contoh kode :
  3. Id Selector
    4. Selektor id merupakan selektor yang bersifat unik. Artinya, selektor
    id hanya dapat kita gunakan untuk satu elemen saja.
    Untuk membuat selector id kita perlu memulainya dengan tanda pagar (#).
  4. Class Selector
    5. Selektor class hampir sama dengan selektor id, bedanya selektor ini
    tidak bersifat unik dan untuk membuatnya kita memerlukan tanda titik (.).
  5. Atribut Selector
    6. Terakhir, atribut selector merupakan selector yang memilih elemen berdasarkan
    tag dan atributnya. Jadi selektor ini hampir sama dengan selektor tag hanya saja
    atribut dari tag yang dipilih ikut didefinisikan.
