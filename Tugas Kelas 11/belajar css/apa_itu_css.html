Apa itu CSS ?

Cascading Style Sheets, atau yang disingkat CSS ini adalah bahasa dalam pemrograman
untuk menentukan bagaimana dokumen itu disajikan. CSS berfungsi untuk menjelaskan dan
menata tampilan elemen yang tertulis pada bahasa markup, salah satunya adalah HTML. Dalam
arti lain, CSS-lah yang menjelaskan bagaimana elemen HTML di balik layar ditampilkan
sedemikian rupa di layar website-mu nanti. Baik HTML maupun CSS, keduanya saling melengkapi.
